About PseudoDojo

Welcome to the PseudoDojo website, a digital hub for pseudopotentials (or 'pseudos,' for short) generated through the PseudoDojo project, a Python package for the development, validation, and storage of pseudopotentials. The pseudopotential repository, including previous versions of pseudopotentials, can be found on our GitHub.

The web interface provides several files:

  • psp8 : a norm-conserving (NC) pseudopotential in psp8 format (Abinit)
  • upf : a NC pseudo in upf format (Quantum ESPRESSO)
  • psml : a NC pseudo in psml format (SIESTA)
  • xml : (PAW) the paw dataset in xml format (Abinit and GPAW)
  • html : a webpage, generated from the corresponding Jupyter notebook, containing the PseudoDojo test report of a NC pseudo.
  • djrepo : a JSON document with the consolidated test report of the NC pseudo.
